BLOG POST IMAGES
=================
Drop blog post thumbnail images here.

  blog-1.jpg   → Blog post #1 thumbnail
  blog-2.jpg   → Blog post #2 thumbnail
  blog-3.jpg   → Blog post #3 thumbnail

To wire up a blog image, add an img property to the post in BLOG_POSTS array:
  {id:1, e:"🏭", img:"images/blog/blog-1.jpg", tag:"News", title:"...", ...}

Recommended: 800 × 500 px · JPG or WEBP · under 200 KB
