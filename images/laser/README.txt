LASER CUTTING — Product Images
================================
Drop your images into this folder with EXACTLY these filenames.
Supported formats: JPG, JPEG, PNG, WEBP

  laser-mdf-shapes.jpg        → Product #1  — Laser Cut MDF Shapes
  acrylic-laser-cut.jpg       → Product #2  — Acrylic Laser Cut Shapes
  acrylic-keychains.jpg       → Product #3  — Custom Acrylic Keychains

TIPS:
• Recommended size: 800 × 600 px or larger (4:3 ratio)
• Keep file size under 300 KB for fast loading
• Use a plain/light background for clean product shots
• You can add multiple images per product:
    acrylic-keychains-2.jpg, acrylic-keychains-3.jpg, etc.
  then add them to the imgs[] array in the PRODUCTS data inside page 1.html
