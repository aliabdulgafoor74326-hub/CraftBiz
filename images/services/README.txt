SERVICE PAGE IMAGES
===================
Each subfolder holds banner/gallery images for that service's dedicated page.

  laser/       → Laser Cutting service page
  cnc/         → CNC Milling service page
  3d/          → 3D Printing service page
  tshirt/      → T-Shirt Printing service page
  gifts/       → Corporate Gifts service page
  homedecor/   → Home Decor service page
  wedding/     → Wedding Cards service page
  resin/       → Resin & Art service page
  stationery/  → Stationery & ID service page

See README.txt inside each folder for exact filenames.

IMAGE SPECS
-----------
• Hero/banner: 1200 × 800 px minimum · JPG or WEBP · under 300 KB
• Gallery:     800 × 600 px minimum · JPG or WEBP · under 200 KB
